
### Arquitetura

Esse projeto Vue foi desenvolvido com a finalidade de consumir uma API NodeJs
podendo listar, excluir, adicionar e editar estudantes. A arquitetura utilizada
foi baseada na arquitetura flux, não fideliza todos os conceitos e técnicas da
arquitetura, mas tem uma iniciativa de se ter um projeto escalável.

### Bibliotecas

Para um desenvolvimento mais ágil e a fim de evitar hardCode foi utilizado algumas bibliotecas de terceiros como axios para as requisições para a API NodeJs, v-mask para máscaras em alguns campos numéricos, utilizado o vuetify para a implementações da UI e vue-sweetalert2 para exibição de mensagens de sucesso e erros.

### Melhorias

Acredito que com mais algumas horas de trabalho nesse projeto, é possível estudar um melhor tratamento para os erros pois sei que é possível melhorar, pode ser melhor abstraído alguns componentes para ficar mais genérico o possível a fim de utilizar em outras pages para outros módulos futuros, e a última mas não menos importante seria instalar e configurar a aplicação com typescript, não tenho conhecimento como o typescript se comporta com Vue, mas isso pode ser resolvido ligeiramente.
